# Libasm

19 Coding School project (42 Network)

>The aim of this project is to get familiar with assembly language.

## Status

Succes: 100%

## How to

Compile and create the archive libasm.a:
```
make
```
Delete all objects:
```
make clean
```
Delete all objects and the archive:
```
make fclean
```
